# Add System

A system may be admitted to SPEEDKIT only if it is a bounded public system with stable recognition metadata.

## Required fields

- system id
- canonical system name
- registry class
- registry status
- discovery URL
- entry point URL
- authoritative specification URL
- bounded purpose
- execution boundary
- output or proof surface
- termination condition
- access rule
- archive guarantee

## Allowed classes

- recognized
- superseded
- discontinued

## Allowed statuses

- registered
- live
- discontinued

## Admission effects

Admission grants public recognition and routing presence inside SPEEDKIT.

Admission does not grant constitutional authority.
Admission does not grant verifier authority.
Admission does not grant state authority.
Admission does not modify other repository identities.

## Removal

Removal does not exist.

## Supersession

A recognized system may be superseded or discontinued, but its registry record remains.
